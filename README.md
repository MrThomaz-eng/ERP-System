# ⚙️ ProductionFlow – ERP de Monitoramento em Tempo Real

**Um ecossistema robusto de gestão industrial com mais de 10 módulos integrados, desenvolvido sob medida para otimizar a cadeia produtiva e o monitoramento de ativos.**

O **ProductionFlow** foi projetado para resolver a complexidade da gestão de dados em tempo real, transformando métricas brutas em dashboards inteligentes para tomada de decisão estratégica.

---

## 🔒 Acesso ao Projeto
> **Nota técnica:** Por se tratar de um sistema ERP completo com integração a banco de dados e regras de negócio privadas, a versão online disponível via GitHub Pages funciona como um **Live Preview da Interface (UI)**. O backend e o processamento de dados em tempo real estão protegidos.

🔗 **[Ver Interface do Sistema](https://MrThomaz-eng.github.io/ERP-System/)**

---

## 🏗️ Estrutura e Módulos

O sistema é composto por mais de **10 módulos especializados**, incluindo:
- **Dashboard Operacional:** Visão holística da produção em tempo real.
- **Gestão de Estoque/Insumos:** Controle rigoroso de entrada e saída de materiais.
- **Monitoramento de Ativos:** Status de máquinas e alertas de manutenção.
- **Controle de Expedição:** Gestão logística e fluxo de saída.
- **Relatórios Analíticos:** Geração de dados para auditoria e performance.

---

## ✨ Diferenciais Técnicos

- **Interface de Alta Densidade de Dados:** Design focado em usabilidade para ambientes industriais, priorizando clareza e agilidade.
- **Arquitetura Modular:** Sistema escalável que permite a adição de novas funcionalidades sem comprometer o núcleo do ERP.
- **Interatividade Avançada:** Uso intensivo de JavaScript e jQuery para manipulação dinâmica de dados e eventos sem recarregamento de página.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura robusta para sistemas complexos.
- **CSS3**: Estilização avançada com foco em painéis de controle (Dashboards).
- **JavaScript (ES6+) & jQuery**: Core da inteligência do sistema e manipulação do DOM.
- **Font Awesome**: Ícones técnicos para identificação de módulos.
- **Google Fonts**: Tipografia focada em legibilidade de dados.
